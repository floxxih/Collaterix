## Contributing

Interested in becoming a maintainer? Start by:
1. Contributing quality PRs
2. Helping with code reviews
3. Assisting community members
4. Maintaining documentation

We recognize consistent, high-quality contributors and may invite them to join the core team.

## Responsibilities

Maintainers are responsible for:
- Reviewing and merging pull requests
- Triaging issues
- Maintaining code quality standards
- Guiding project direction
- Community engagement
